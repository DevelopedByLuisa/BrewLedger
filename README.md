# BrewLedger

BrewLedger is a simple and efficient solution for managing coffee capsules (or similar consumable supplies) in shared environments such as offices or coworking spaces.

The tool helps accurately track each user's consumption, ensuring a fair and transparent cost allocation.

## The Problem

In shared environments where multiple people consume common resources (such as coffee capsules), it can be difficult to keep track of inventory while ensuring that everyone pays their fair share.

BrewLedger automates this process by connecting physical consumption with digital accounting.

## Features

- **Inventory Management**  
  Monitor the current stock of coffee capsules and keep track of remaining inventory.

- **User Consumption Tracking**  
  Record and track the consumption of each individual user.

- **Automated Cost Calculation**  
  Automatically calculate each user's costs based on the configured unit price (e.g., per coffee capsule).

- **Clear Consumption Overview**  
  Generate an easy-to-understand overview of who consumed what, making monthly or quarterly billing straightforward and transparent.
